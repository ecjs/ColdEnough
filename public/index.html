About This Project

I created this as a simple project whilst learning Javascript/jQuery/Node JS. It answers a simple question, using geolocation or manual city selection.

The site uses the Weather Underground API to produce temperature information based on location. We also use Geobytes.com for city and state prepopulation for the input field.

A special thanks to jQuery and Bootstrap for making this a lot easier.
